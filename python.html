[문제] 다음 Python 프로그램의 실행 결과를 쓰시오.

class City:
    def __init__(self):
        self.cities = ["Seoul", "Busan", "Daegu", "Incheon"]
    
    def get_initials(self):
        result = ""
        for c in self.cities:
            result += c[0]
        return result

obj = City()
print(obj.get_initials())
정답 보기
SBDI
해설 보기
  1. "Seoul"[0] → 'S'
  2. "Busan"[0] → 'B'
  3. "Daegu"[0] → 'D'
  4. "Incheon"[0] → 'I'

문자열 인덱싱에서 [0]은 첫 번째 문자를 반환합니다. 클래스 내부 리스트의 각 원소에서 첫 글자만 추출하는 패턴입니다.

2021년 2회

[문제] 다음 Python 프로그램의 실행 결과를 쓰시오.

x = 100
result = 0
i = 1
while x > 0:
    x = x >> i
    result += x
print(result)
정답 보기
97
해설 보기

>>는 오른쪽 비트 시프트로, 정수를 2로 나눈 몫과 같습니다.

  1. x = 100 >> 1 = 50, result = 0 + 50 = 50
  2. x = 50 >> 1 = 25, result = 50 + 25 = 75
  3. x = 25 >> 1 = 12, result = 75 + 12 = 87
  4. x = 12 >> 1 = 6, result = 87 + 6 = 93
  5. x = 6 >> 1 = 3, result = 93 + 3 = 96
  6. x = 3 >> 1 = 1, result = 96 + 1 = 97
  7. x = 1 >> 1 = 0, result = 97 + 0 = 97 → x=0이므로 루프 종료

※ 주의: x = x >> i에서 시프트 후의 결과(0 포함)를 result에 더한 뒤 x=0이면 while 조건 검사에서 탈출합니다. 마지막 반복에서 x가 0이 되면 0을 더하고 루프가 끝납니다.

[문제] 다음 Python 프로그램의 실행 결과를 쓰시오.

a = [1, 2, 3]
b = a
b[1] = 99
print(a == b)
print(a is b)
정답 보기
True
True
해설 보기

b = a는 리스트를 복사하지 않고 같은 객체를 참조합니다.

  1. b[1] = 99 → a도 [1, 99, 3]이 됨
  2. a == b → 내용 같음 → True
  3. a is b → 같은 객체 → True

is vs ==: ==는 값 비교, is는 동일 객체(identity) 비교. 리스트는 mutable이라 참조 복사 시 주의해야 합니다.

a = {'한국', '중국', '일본'}

[문제] 다음 Python 프로그램의 실행 결과를 쓰시오.

a = [4, 3, 2, 1]
a.reverse()
b = [a[i] - a[i+1] for i in range(len(a)-1)]
print(sum(b))
정답 보기
-3
해설 보기
  1. a = [4, 3, 2, 1]
  2. a.reverse() → a = [1, 2, 3, 4] (원본 리스트를 직접 뒤집음, in-place)
  3. 리스트 컴프리헨션 b를 계산합니다:
    • i=0: a[0] - a[1] = 1 - 2 = -1
    • i=1: a[1] - a[2] = 2 - 3 = -1
    • i=2: a[2] - a[3] = 3 - 4 = -1
  4. b = [-1, -1, -1]
  5. sum(b) = (-1) + (-1) + (-1) = -3

주의: 오름차순 정렬된 배열에서 a[i] - a[i+1]은 항상 음수입니다. 부호를 놓치지 않도록 주의하세요. reverse()는 sorted()와 달리 새 리스트를 반환하지 않고 원본을 직접 변경합니다(반환값은 None).

2025년 1회

[문제] 다음 Python 프로그램의 실행 결과를 쓰시오.

def calc(n):
    if n <= 1:
        return n
    return calc(n-1) + calc(n-2)

print(calc(7))
정답 보기
13
해설 보기

이것은 피보나치 수열의 재귀 구현입니다.

F(0)=0, F(1)=1, F(2)=1, F(3)=2, F(4)=3, F(5)=5, F(6)=8, F(7)=13

재귀적으로 F(n) = F(n-1) + F(n-2)를 계산합니다. 시간 복잡도는 O(2ⁿ)으로 비효율적이지만, 시험에서는 트레이싱 능력을 테스트합니다.

[문제] 다음 Python 프로그램의 실행 결과를 쓰시오.

a = [1, 2, 3, 4, 5]
b = a[1:4]
b[0] = 99
print(a)
print(b)
정답 보기
[1, 2, 3, 4, 5]
[99, 3, 4]
해설 보기
  1. b = a[1:4] → [2, 3, 4] (슬라이싱은 새로운 리스트를 생성)
  2. b[0] = 99 → b만 변경 → [99, 3, 4]
  3. a는 영향 없음 → [1, 2, 3, 4, 5]

핵심: b = a와 달리 b = a[1:4]는 새 리스트를 만드는 얕은 복사(shallow copy)입니다. 원본에 영향을 주지 않습니다.
